20. contoh soal mtk persentase rugi mtk kelas 7 semester 2


Contoh :

Seorang pedagang membeli 1 buah buku dengan harga 5.000, kemudian pedagang menjualnya dengan harga 4.500. Tentukan lah peresentase kerugian yang di alamai oleh pedagang..

Jawaban :

Diketahui
Harga Beli (HB) = 5.000
Harga Jual (HJ) = 4.500

DItanya %Rugi


Penyelesaian :

➡Tentukan besar rugi :

Besar rugi :
= HB - HJ
= 5.000 - 4.500
= 500.-

Maka, % Rugi :
= (besar rugi / HB) X 100%
= (500 / 5000) x 100%
= (50000/5000) %
= 10%

Jadi persentase rugi = 10%

28. pakai caranya soal matematika kelas 4 SD semester 2 tentang Segitiga​


Rumus keliling segitiga

Keliling segitiga =  a + b + c

Jawaban atas pertanyaan dari segitiga A-E

Pertama-tama, anggap

Kolom ke-2 = a Kolom ke-3 = b dan Kolom ke-4 = c

A. Diketahui = a = 16 cm, b = 15 cm, c =17 cm.

Ditanya = keliling =___?

Keliling segitiga = a + b + c

Keliling segitiga = 16 cm + 15 cm +17 cm = 48 cm.

B. Diketahui = b = 42 dm, c = 46 dm dan keliling = 125 dm.

Ditanya = a =___?

Keliling segitiga = a + b + c

125 dm = a + 42 dm + 46 dm

125 dm - 42 dm - 46 dm = a

83 dm - 46 dm = a

a = 37 dm.

C. Diketahui = a = 20 m, c = 25 m, Keliling = 67 m

Ditanya = b =___?

Keliling segitiga = a + b + c

67 m = 20 m + b + 25 m

67 m - 20 m - 25 m = b

47 m - 25 m = b

b = 22 m.

D. Diketahui = a = 23 cm, b = 26 cm, Keliling = 63 cm

Ditanya = c =___?

Keliling segitiga = a + b + c

63 cm = 23 cm + 26 cm + c

63 cm - 23 cm - 26 cm = c

c = 40 cm - 26 cm = 14 cm.

E. Diketahui = a = 29 mm, b = 32 mm, Keliling = 100 mm

Ditanya = c =___?

Keliling segitiga = a + b + c

100 cm = 29 cm + 32 cm + c

c = 100 cm - 29 cm - 32 cm.

c = 71 cm - 32 cm = 39 cm.

1.) Diketahui :

jarak pusat lingkaran AB = 10 cm

jari-jari A = 11 cm

jari-jari B = 3 cm

Ditanya :

a. panjang garis singgung persekutuan luar

b. sketsa gambar garis singgung persekutuan luar

Jawab :

a. panjang garis singgung persekutuan luar

garis singgung persekutuan luar kita beri nama CD

CD² = AB² - (AD - BC)²

= 10² - (11 - 3)²

= 10² - 8²

= 100 - 64

= 36

CD = √36

= 6 cm

Jadi panjang garis singgung persekutuan luar adalah 6 cm

b. Sketsa gambar garis singgung persekutuan luar bisa dilihat diatas

2.) a] Jarak pusat = √(garis singgung² + (R - r)²)

Jarak pusat = √(576 + 49)

Jarak pusat = √(625)

Jarak pusat = 25 cm

b] Jarak kedua lingkaran

= jarak pusat - (R + r)

= 25 - (15 + 8)

= 25 - 23

= 2 cm

4.) Diketahui :

d₁ + d₂ = 30 cm

garis singgung persekutuan luar (d) = 24 cm

jarak pusat kedua lingkaran (p) = 26 cm

a. jari-jari kedua lingkaran tersebut

Untuk gambar bisa dilihat pada lampiran

jumlah jari-jari kedua lingkaran

d₁ + d₂ = 30

2R + 2r = 30 (kesemua ruas dibagi 2)

R + r = 15 ... pers I

selisih jari-jari kedua lingkaran

d² = p² - (R - r)²

24² = 26² - (R - r)²

576 = 676 - (R - r)²

(R - r)² = 676 - 576

(R - r)² = 100

R - r = √100

R - r = 10 ... pers II

eliminasi pers I dan II
