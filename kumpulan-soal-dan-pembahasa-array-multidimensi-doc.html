2. jelaskan definisi dari array multidimensi dan berikan contoh array multidimensi yang dideklarasikan dan di gunakan dalam perhitungan?​


Array multidimensi adalah kumpulan data yang tersusun dalam bentuk matriks atau tabel dua dimensi, atau lebih dari dua dimensi. Setiap elemen dalam array multidimensi dapat diakses melalui indeks yang terdiri dari beberapa nilai, tergantung pada dimensi array tersebut.

Contoh array multidimensi yang dideklarasikan dan digunakan dalam perhitungan adalah array 2x3 yang menyimpan nilai-nilai dalam matriks:

int[][] matrix = {{1, 2, 3}, {4, 5, 6}};

Dalam contoh ini, kita mendeklarasikan array multidimensi dengan 2 baris dan 3 kolom. Setiap elemen dalam matriks ini dapat diakses menggunakan indeks dua dimensi. Misalnya, untuk mengakses nilai 4, kita dapat menggunakan matrix[1][0] karena 1 adalah indeks baris dan 0 adalah indeks kolom.

4. perbedaan array dua dimensi dan multidimensi


Pengertian Array Dua Dimensi
Array dua dimensi dapat dipandang sebagai gabungan array satu dimensi.
Array dua dimensi merupakan array yang terdiri dari m buah baris dan n buah kolom.
Bentuknya dapat berupa matriks atau tabel.
Deklarasi array :
Tipe_array nama_array[baris][kolom];
Contoh :
Int X[3][4];
Pengertian Array Multidimensi
Array multi-dimensi merupakan array yang mempunyai ukuran lebih dari dua.
Bentuk pendeklarasian array sama saja dengan array dimensi satu maupun array dimensi dua.
Bentuk umumnya yaitu :
tipe_array nama_array[ukuran1][ukuran2]…[ukuranN];
Contoh :
float X[2][4][3];

7. Soal :  Diketahui ada 3 buah array 2D yaitu A, B, C dengan dimensi masing-masing 4x5  Buatlah program untuk menjumlahkan setiap isi sel array A dengan B, hasilnya disimpan dalam array C.tolong di antu KK,bahasa pemrogramannya c++​


Untuk menjumlahkan setiap isi sel array A dengan B dan menyimpannya dalam array C, kita dapat menggunakan perulangan (loop) untuk mengelilingi setiap elemen dalam array A dan B. Setelah itu, kita dapat menambahkan setiap elemen dari array A dengan elemen yang sama di array B dan menyimpannya di array C pada indeks yang sama. Berikut ini adalah contoh implementasi program untuk menjumlahkan setiap isi sel array A dengan B dan menyimpannya dalam array C:

# deklarasi array A, B, dan C

A = [[0 for i in range(4)] for j in range(5)]

B = [[0 for i in range(4)] for j in range(5)]

C = [[0 for i in range(4)] for j in range(5)]

# inisialisasi isi array A dan B

for i in range(4):

for j in range(5):

A[i][j] = i + j

B[i][j] = i * j

# jumlahkan setiap elemen dari array A dan B

# dan simpan hasilnya dalam array C

for i in range(4):

for j in range(5):

C[i][j] = A[i][j] + B[i][j]

# cetak isi array C

for i in range(4):

for j in range(5):

print(C[i][j], end=" ")

print()

Catatan: Kodisi di atas merupakan contoh implementasi program untuk menjumlahkan setiap isi sel array A dengan B dan menyimpannya dalam array C. Anda dapat mengubah dan menyesuaikan kode tersebut sesuai dengan kebutuhan Anda.

26. diketahui sebuah array = {10,30,25,50,60,...} dan stok = 80. gimana cara buat array multidimensi untuk menyimpan masing-masing nilai di atas berdasarkan sisa stok yg ada. kalau stok sudah tidak mencukupi, biar masuk ke array baru dan stok kembali full jadi 80 lagi gitu, terus berulang sampe semua nilai dalam array di awal tadi terpilih semua. bhs pemrograman apa aja bebas


#include <iostream>

#include <vector>

using namespace std;

int main(){

int arr[] = {50,70,45,80,30,35,60,75,55,95};

int n = sizeof(arr)/sizeof(int);

int stock = 150;

vector<vector<int>> v;

 

int currIdx = 0;

int currStock = stock;

int it = 0;

v.push_back({});

while (currIdx < n){

 if (arr[currIdx] <= currStock){

  v[it].push_back(currIdx);

  currStock -= arr[currIdx];

  currIdx++;

 }

 else{

  v.push_back({});

  if (arr[currIdx] > stock) currIdx++;

  it++;

  currStock = stock;

 }

}

 

for (int i = 0; i < v.size(); i++){

 printf("Warehouse %d : ", i);

 for (int j = 0; j < v[i].size(); j++){

  printf("%d ", v[i][j]);

 }

 printf("\n");

}

 

return 0;

}

40. kelebihan dan kekurangan menggunakan array multidimensi


Keunggulan array adalah sebagai berikut:

1. Array sangat cocok digunakan untuk pengaksesan acak (random). Karena sembarang elemen di array dapat diacu secara langsung tanpa melalui elemen-elemen lain.

2. Jika berada di suatu lokasi elemen, maka sangat mudah untuk menelusuri ke elemen-elemen tetangga, baik elemen sebelumnya ataupun elemen berikutnya.

3. Jika elemen-elemen array adalah nilai-nilai independen dan seluruhnya harus terjaga, maka penggunaan penyimpanannya sangatlah efisien.


Kelemahan array adalah sebagai berikut:

1. Array selalu bertipe homogen. Kita tidak dapat mempunyai array diaman satu elemen adalah karakter, elemen lain bilangan dan elemen lain adalah tipe-tipe data yang lainnya yang berbeda.
