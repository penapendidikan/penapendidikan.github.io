Uraian:

1. Apa yang kamu ketahui tentang empati?

=> Empati adalah keadaan mental yang membuat orang merasa dirinya dalam keadaan, perasaan atau pikiran yang sama dengan orang lain.

2. Mengapa kita harus memiliki sikap empati?

=>Kita harus memiliki empati karena merupakan sifat terpuji. Islam sangat menganjurkan sikap empati, sebagaimana firman Allah SWT dalam Q.S. an-Nisa/4: 8.

3. Mengapa kita harus menghormati orang tua?

=>Jasa orang tua sangat besar sehingga sebagai anak wajib hukumnya berbuat baik kepada keduanya.

4. Bagaimana caranya menghormati orang tua kita yang masih hidup?

=>Memperlakukan keduanya dengan sopan dan hormat;

=>Membantu pekerjaanya;

=>Mengikuti nasihatnya (apabila nasihat itu baik);

=>Membahagiakan keduanya.

5. Bagaimana caranya menghormati orang tua kita yang sudah meninggal?

=>Jika keduanya muslim, dapat mendoakan mereka setiap saat agar mendapat ampunan Allah Swt;

=>Melaksanakan wasiatnya;

=>Menyambung dan melanjutkan silaturahmi yang dahulu sudah dilakukan oleh kedua orang tua;

=>Menjaga nama baik mereka.

6. Siapakah guru itu? Dan mengapa kita harus menghormatinya?

=>Guru telah mendidik dan mengajarkan ilmu kepada kita. Sebagai pendidik, guru membentuk kita menjadi manusia yang beriman, mengerti baik dan buruk, berbudi pekerti luhur, dan menjadi orang yang bertanggung jawab, baik kepada diri sendiri, masyarakat, bangsa, maupun negara.

=>Berlatih mengorbankan milik sendiri

=>Membahagiakan orang lain

8. Buatlah contoh perilaku menghormati kedua orang tua!

Contoh perilaku menghormati kedua orang tua yakni:

=>Membantu pekerjaan rumah, seperti menyapu lantai dan mencuci piring

=Bertutur kata yang baik, sopan dan santun saat berbicara dengan keduanya

=>Mengikuti nasihat dan perintah orang tua.

9. Buatlah contoh perilaku menghormati guru!

=>Melaksanakan nasihatnya

=>Memperhatikan apabila diajak bicara di kelas

40. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5
