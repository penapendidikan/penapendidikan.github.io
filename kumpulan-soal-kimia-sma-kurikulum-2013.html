Pengelompokan Atom Karbon

Berdasarkan kedudukannya pada rantai senyawa hidrokarbon, Atom karbon dikelompokkan menjadi beberapa jenis :  

Atom C yang mengikat 1 atom C yaitu Atom C Primer  Atom C yang mengikat 2 atom C lainnya yaitu Atom C Sekunder  Atom C yang mengikat 3 atom C lainnya yaitu Atom C Tersier  Atom C yang mengikat 4 atom C lain yaitu Atom C Kuartener.

ALKANA  

Alkana yaitu beberapa dari senyawa hidrokarbon yang memiliki rumus umum CₙH₂ₙ₊₂. Tatanama alkana diberi dengan akhiran -ana dan merupakan hidrokarbon tak jenuh dengan ikatan rantai tunggal.

Berikut adalah penjelasan dari masing-masing jawaban di atas:

1. Bahan kimia dapat kita temukan di sekeliling kita. Bahkan tubuh kita juga terbentuk dari bahan-bahan kimia, sehingga tidak semua bahan kimia itu berbahaya. Namun, penyedap rasa tidak mengandung zat gizi yang dibutuhkan untuk tumbuh dan berkembangnya tubuh, sehingga opsi C benar.

6. Perumusan masalah merupakan hal yang ingin dicari jawabannya melalui eksperimen kimia. Sehingga, opsi B benar.

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}
