9. Buatlah flowchart algoritma untuk menukar isi dari variable A dan variable B, dimana variable A=3 dan variable B=2


Jawaban:

Deklarasi variable A = 3, dan B = 2Deklarasi variable temp;Simpan nilai A dalam temp, yaitu temp = A;Simpan nilai B dalam A, yaitu A = B;Lalu simpan kembali nilai temp dalam B, yaitu B = temp;

Penjelasan:

Salah satu cara untuk menukar nilai variable ialah untuk menggunakan variable lain untuk menampung nilai dari salah satu variable tersebut. Disini saya menggunakan variable temp atau temporary (penamaan variable itu bebas, tidak harus memakai temp).

Jadi simplenya saya hanya menyimpan nilai A dalam temp, lalu saya tindih nilai A dengan B, dari situ A = 2, B = 2, dan temp = 3. Setelah itu tinggal kita tindih lagi nilai di dalam B dengan temp, dari situ A = 2, dan B = 3.

34. diketahui persamaan 5x + y = 9, jika variable y dinyatakan dalam x maka y =...​


Untuk menampilkan variabel y dalam persamaan 5x + y = 9 sebagai fungsi dari x, kita perlu memindahkan variabel x ke sisi lain persamaan dan mengubah tanda positif menjadi negatif, sehingga kita memiliki:

y = 9 - 5x

Sehingga jika kita memiliki nilai x, kita dapat menggunakan persamaan ini untuk menghitung nilai y yang sesuai.

1. Variabel Bebas (Variabel Independen)

Variabel Bebas atau juga dikenal dengan istilah variabel Independent adalah merupakan variabel yang diduga memiliki fungsi sebagai penyebab timbulnya variabel yang lain. Biasanya variabel bebas akan dimanipulasi, diamati dan diukur dengan tujuan untuk mengetahui sejauh mana pengaruhnya terhadap variabel lainnya. Secara singkat, variabel bebas adalah variabel yang dapat mempengaruhi atau menjadi penyebab perubahan atau timbulnya variabel dependen (terikat). Variabel bebas memiliki fungsi utama sebagai acuan untuk mengetahui pengaruhnya terhadap variabel lain.
