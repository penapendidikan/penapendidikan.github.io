5. 3.Nyatakan perbandingan Trigometri dari segitiga siku siku berikut dalam sudut (x ß dan y). (perbandingan sudut yang di maksud adalah sin, cos, tan, Cotg, sec,cosec )soal nya ada 3 bangun kakmakasi kak/pak/bu Yang udah bantu jawab soal nya esok di kumpul​​


Jawaban:

Soal a

Sisi yang menghadap sudut α adalah y. Sisi di samping sudut α adalah x. Sisi miringnya adalah z.

Oleh karena itu, perbandingan trigonometrinya pada ∠α adalah:

sin α = y/zcos α = x/ztan α = y/xcot α = 1/(tan α) = x/ysec α = 1/(cos α) = z/xcosec α = 1/(sin α) = z/y

Soal b

Panjang sisi miring belum diketahui. Oleh karena itu, kita hitung terlebih dahulu dengan teorema Pythagoras:

z² = x² + y²

Di sini saya gunakan variabel yang sesuai dengan soal a di atas, agar bisa lebih mudah dipahami. Perbandingan trigonometri ∠β nanti akan merujuk variabel seperti pada soal a.

y = 12  ← sisi yang menghadap ∠β

x = 5  ← sisi di samping ∠β

⇔ z² = 5² + 12²

⇔ z² = 25 + 144 = 169

⇔ z = √169

⇔ z = 13

Sehingga, perbandingan trigonometrinya pada ∠β adalah:

sin β = 12/13cos β = 5/13tan β = 12/5cot β = 5/12sec β = 13/5cosec β = 13/12

Soal c

Pada segitiga tersebut, panjang sisi yang menghadap sudut γ (sisi y) belum diketahui.

Kita hitung dulu dengan teorema Pythagoras.

z² = y² + x²

z = 5   ← sisi miring

x = 4   ← sisi di samping sudut γ

5² = y² + 4²

⇔ y² = 5² – 4²

⇔ y² = 25 – 16 = 9

⇔ y = √9

⇔ y = 3

Sehingga, perbandingan trigonometrinya pada ∠γ adalah:

sin γ = 3/5cos γ = 4/5tan γ = 3/4cot γ = 4/3sec γ = 5/4cosec γ = 5/3

20. Given that 90° < x < 180° and Tan(x) = -3, determine the value of Sin(x) and Cos(x).​MOHON DI BANTU KK ABG SOAL MYA BESOK MAU SK KUMPUL


Mengingat 90° < x < 180° dan Tan(x) = -3, kita dapat menentukan nilai Sin(x) dan Cos(x) menggunakan sifat fungsi tangen, sinus, dan cosinus.

Kita tahu bahwa Tan(x) = Sin(x) / Cos(x). Karena Tan(x) = -3, kita dapat mensubstitusikannya ke dalam persamaan:

-3 = Sin(x) / Cos(x)

Untuk memecahkan Sin(x), kita dapat mengatur ulang persamaan:

Sin(x) = -3 * Cos(x)

Karena Cos(x) ≠ 0 (karena Tan(x) terdefinisi dan tidak sama dengan tak terhingga), kita dapat membagi kedua sisi persamaan dengan Cos(x):

Sin(x) / Cos(x) = -3

Sisi kiri persamaan mewakili fungsi tangen, sehingga kita dapat menulis:

Tan(x) = -3

Sekarang, kita dapat mencari nilai x dengan mengambil arctan (-3):

x = arctan(-3)

Menggunakan kalkulator, menemukan bahwa x ≈ 116,565°.

Sekarang setelah mengetahui nilai x, kita dapat mencari Sin(x) dan Cos(x) menggunakan fungsi sinus dan cosinus:

Sin(x) = Sin(116,565°) ≈ 0,9239

Cos(x) = Cos(116,565°) ≈ -0,3827

Oleh karena itu, nilai Sin(x) adalah sekitar 0,9239 dan nilai Cos(x) adalah sekitar -0,3827.
