Sistem Persamaan Linear Dua Variabel adalah persamaan garis yang memuat dua variabel dengan masing-masing berpangkat tertinggi satu.

Bentuk Umum :

ax + by = c

Keterangan :

a dan b = Koefisien

x dan y = Variabel

c = Konstanta

Untuk mendapatkan penyelesaian dari SPLDV, dapat menggunakan beberapa metode sebagai berikut :

Metode SubstitusiMetode EliminasiMetode Campuran (Eliminasi & Substitusi)Metode Grafik

3x + 4y = -2

↪ 3x = -2 - 4y

↪ x = [tex]\sf \frac{-2 - 4y}{-3}[/tex]

Substitusi ke persamaan awal

↪ 3x + 4y = -2

↪ 3( [tex]\sf \frac{-2 - 4y}{-3}[/tex]) + 4y = -2

↪ -(-2 - 4y) + 4y = -2

↪ 2x + 4y + 4y = -2

↪ 2 + 8y = -2

↪ 8y = -2 - 2

↪ 8y = -4

↪ y = [tex]\sf -\frac{4}{8}[/tex]

↪ y = [tex]\sf \red{- \frac{1}{2}}[/tex]

Substitusi nilai y ke persamaan awal
