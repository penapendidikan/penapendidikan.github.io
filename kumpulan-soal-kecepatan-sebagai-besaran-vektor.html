r(2) = 14 i + 17 j

t = 4s
~ r(t) = (2t²+3t) i + (1+2t³) j
r(4) = (2.4²+3.4) i + (1+2.4³) j
r(4) = 44 i + 129 j

- vektor perpindahan
Δr = r₂-r₁
= (44-14) i + (129-17) j
Δr = 30 i + 112 j

- besar perpindahan
|Δr| = √(30)²+(112)²
= √900+12.544
= √13.444
= 115,94 meter

- arah perpindahan
tan Ф = Δy/Δx
= 112/30 = 3,733
Ф = 75,00
