Luas Segitiga = 1/2 x a x t
=1/2 x 36 x 16
=288 cm Pangkat duaRumus luas segitiga=1/2×a×t
=1/2×36cm×16cm
=288cm2

Jadi luas segitiga tersebut adalah 288cm2

Jawab:

1. lingkaran

k = π x d

L = π x r²

2. persegi

K = 4 x s

L = s x x

3. persegi panjang

K = 2 x (p + l)

L = p x l

4. segitiga

K = sisi 1 + sisi 2 + sisi 3

L = ½ x a x t

5. layang layang

K = sisi a + b + c + d

L = ½ x diagonal 1 x diagonal 2

6. belah ketupat

K = 4 x s

L = ½ × d1 × d2

7. jajar genjag

K = 2 x panjang + 2 x lebar atau bisa ditulis dengan 2 (panjang + lebar)

L =  a x t

8. trapesium

K = a + b + c + d

L =  1/2 x (a + b) x t

9. kubus

L = 6  x s²

V = s x s x s

10. balok

L = 2 x (p x l + p x t + l x t)

V = p x l x t

11. limas

L = L alas x jumlah luas bidang tegak segitiga

V = 1/3 x luas alas x t limas

12. prisma

L = (2 x luas alas) + (keliling alas x tinggi prisma)

V = luas alas x t prisma

13. tabung

L = 2 x π x r (r + t)

V = π x r² x t

14. kerucut

L = π x r x (r + s)

V = 1/3 x luas alas x t

15. bola

L = 4 x π x r²

V = 4/3 x π x r3
