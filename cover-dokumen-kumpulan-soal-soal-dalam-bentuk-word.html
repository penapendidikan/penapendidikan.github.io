Magna Carta adalah hasil dari perselisihan antara Paus, Raja John, dan baronnya atas hak-hak raja: Magna Carta mengharuskan raja untuk membatalkan beberapa hak dan menghargai beberapa prosedur legal, dan untuk menerima bahwa keinginan raja dapat dibatasi oleh hukum. 1.Raja beserta keturunannya berjanji akan menghormati kemerdekaan, hak, dan kebebasan Gereja Inggris.

3.Para petugas keamanan dan pemungut pajak akan menghormati hak-hak penduduk.

4.Polisi ataupun jaksa tidak dapat menuntut seseorang tanpa bukti dan saksi yang sah.

5.Seseorang yang bukan budak tidak akan ditahan, ditangkap, dinyatakan bersalah tanpa perlindungan negara dan tanpa alasan hukum sebagai dasar tindakannya.

11. Sebutkan cara menduplikat atau memindahkan dokumen pada microsoft word INI PELAJARAN TIK SOALNYA TIK GA ADA PELAJARANYA DI BRAINLY MOHON BANTUANYA SECEPATNYA


Untuk dokumen pendek dengan satu halaman atau kurang, cara tercepat untuk menyalin halaman adalah Pilih Semua , lalu salin.
1. Tekan Ctrl + A di keyboard Anda untuk menyoroti semua teks di dokumen.
Tips: Anda juga dapat menyoroti seluruh dokumen dengan menempatkan kursor mouse di margin kiri, lalu langsung klik tombol mouse kiri tiga kali berturut-turut.
2. Tekan Ctrl + C untuk menyalin seluruh pilihan yang disorot.
Catatan: Ingatlah bahwa fungsi akan menyoroti semua yang ada di dokumen Anda. Metode ini paling cocok digunakan untuk menyalin dokumen yang terdiri dari satu halaman saja.
Menyalin halaman di dokumen dengan beberapa halaman
Jika dokumen Word Anda memiliki beberapa halaman, cara terbaik untuk menyalin satu halaman adalah secara manual memilih dan menyalin teks yang diinginkan.
1. Letakkan kursor Anda di awal halaman yang ingin disalin.
2. Klik dan seret kursor ke bagian bawah halaman yang ingin Anda salin.
3. Tekan Ctrl + C di keyboard Anda.
Tips: Cara lain untuk menyalin teks yang disorot adalah dengan mengklik Beranda > Salin .
Menyisipkan teks yang disalin ke halaman lain
Setelah menyalin konten halaman, Anda dapat menempelkannya ke halaman lain di dokumen atau ke dokumen baru yang dibuat.
1. Tempatkan kursor Anda di bagian bawah halaman, sebelum halaman yang disalin.
2. Klik Sisipkan > Halaman Kosong untuk menambahkan halaman baru.
Catatan: Hentian halaman juga mengakhiri halaman saat ini dan berpindah ke halaman berikutnya. Tekan Ctrl + Enter di keyboard Anda untuk membuat hentian halaman.
3. Letakkan kursor tempat Anda ingin teks agar muncul di halaman baru. Tekan Ctrl + V di keyboard Anda untuk menempelkan teks yang disalin ke halaman.
Catatan: Ketika menempelkan ke halaman baru, Anda memiliki opsi untuk mempertahankan pemformatan yang sama atau menempelkan teks tanpa pemformatan. Gunakan Opsi Tempel yang muncul di bawah teks yang disalin untuk memilih opsi pemformatan Anda.
Menyalin halaman ke dokumen baru
Untuk membuat dokumen baru, klik File >
Baru > Dokumen kosong, tempelkan konten ke dokumen baru, lalu simpan.

21. File dokumen yang diketik menggunakan Microsoft Word, saat disimpan dalam bentuk format


File dokumen yang dibuat menggunakan Microsoft Word, saat disimpan dalam bentuk format .doc atau .docx

Pembahasan:

Microsoft word adalah salah satu software aplikasi pengolah kata yang sangat populer di seluruh dunia. Software ini memiliki konsep sebagai program pengolah kata dengan tampilan grafis yang dapat membuat dokumen dengan hasil cetak sesuai dengan apa yang diketik oleh pengguna. Microsoft word merupakan bagian dari paket aplikasi Microsoft Office yang di dalamnya juga terdapat Microsoft Excel, Microsoft PowerPoint, Microsoft Outlook, dan lain-lain. Microsoft Office sendiri merupakan produk dari perusahaan IT besar di dunia, yaitu Microsoft.

Dari fungsinya yang merupakan aplikasi pengolah kata, Microsoft Word dapat digunakan untuk:

Membuat surat menyuratMembuat dokumenMembuat laporanMembuat grafik, dan lain-lainnya yang berhubungan dengan pengolahan kata, termasuk juga angka.

Fungsi dari Microsoft Word menjadikan software aplikasi ini salah satu aplikasi pengolah kata yang mudah dan simpel untuk digunakan. Tidak hanya kemudahan yang ditawarkan, fitur-fitur untuk membantu pengguna dalam menyelesaikan pekerjaan yang berhubungan dengan pengolahan kata atau angka juga dapat digunakan untuk membuat berbagai dokumen, data, dan lain-lainnya dalam satu tampilan. Hampir semua kalangan menggunakan aplikasi ini untuk mengolah kata karena memang menjadi aplikasi yang termasuk dalam paket pembelian Windows ketika membeli gadget komputer atau laptop baru. Namun tidak hanya di Windows, Microsoft Word juga dapat digunakan di MacOS (Apple), Linux, dan beberapa software gadget lainnya.

Selain itu, File dari Microsoft Word juga dapat dibuka di aplikasi Microsoft lainnya dengan mudah, tetapi tentu memerlukan beberapa penyesuaian karena berbeda format file. Format bawaan atau standard penyimpanan dari Microsoft Word adalah .doc atau .docx. Arti dari format file ini adalah document yang menyimpan hasil dari pekerjaan ketika kita menggunakan Microsoft Word. Namun, masih banyak beberapa format file yang dapat kita pilih untuk menyimpan hasil pekerjaan kita, beberapa format file yang dapat disimpan menggunakan Microsoft Word yaitu:

Word 97-2003 documentWord templatePDFXPS Document\Web pageSingle file web pagePlain textRich text formatOpenDocument text, dan lain-lain

30. Header dan Footer adalah .... a. Cover sebuah dokumen b. Nomor halaman dokumen c. keterangan disetiap halaman dokumen d. keterangan di cover dokumen


Header dan Footer adalah keterangan disetiap halaman dokumen (C)

Pembahasan:

Secara garis besar, header dan footer merupakan keterangan di setiap halaman. Yang mebedakan keduanya adalah letaknya. Header merupakan keterangan pada setiap halaman pada suatu dokumen. Header biasanya berupa judul karya tulis atau judul buku. Letak headernya di atas setiap lembar kerja. Sedangkan footer adalah keterangan di bawah dokumen, biasanya merupakan judul bab.

Untuk membuat header adalah buka dokumen yang akan diberi Header, kemudian klik menu insert, lalu pilih Ikon Header. Sedangkan untuk membuat footer adalah cara membuatnya serupa dengan Header, namun ikon yang di pilih adalah ikon Footer.

Microsoft Word menawarkan galeri halaman sampul siap pakai yang nyaman. Pilih halaman sampul dan ganti teks sampel dengan konten Anda sendiri.

Pada tab Sisipkan, di grup Halaman, klik Halaman Sampul.

pita office 14

Klik tata letak halaman sampul dari galeri opsi.

Setelah Anda menyisipkan halaman sampul, Anda bisa mengganti teks sampel dengan teks Anda sendiri dengan mengklik untuk memilih area dari halaman sampul, seperti judul, dan mengetik teks Anda.

Catatan:

Jika Anda menyisipkan halaman sampul lain dalam dokumen, halaman sampul baru akan menggantikan halaman sampul pertama Anda sisipkan.

Untuk mengganti halaman sampul yang dibuat di versi Word yang lebih lama, Anda harus menghapus halaman sampul pertama secara manual, lalu menambahkan halaman sampul dengan desain dari galeri Word.

Untuk menghapus halaman sampul yang disisipkan dengan Word, klik tab Sisipkan, klik Halaman sampul di grup Halaman, lalu klik Hapus Halaman Sampul Saat Ini.


37. untuk memberikan cover pada dokumen yang kita buat adalah ​


Jawaban:

Pada Insert tab, Pages group, klik Cover Page untuk menampilkan galeri template cover page. Klik salah satu template yang diinginkan. Cover page akan otomatis ditempatkan di halaman pertama dokumen.  Jika ingin menempatkannya di halaman lain: pada galeri klik kanan thumbnail cover page dan klik pilihan yang diinginkan. Insert Cover Page di Halaman Lain Isi informasi pada field di cover page seperti judul, sub judul, dan lain-lain. Untuk membuat tanggal, klik tanda panah dan pilih tanggal pada kalender. Beri gaya format yang diinginkan (bold, italic, underline) atau ganti warna dan ukuran font. Beberapa template cover page menyediakan gambar, bila ingin menggantinya, klik kanan pada gambar dan pilih Change Picture. Mengganti Gambar Cover Page Bila tidak tersedia gambar, kita dapat menambahkan gambar dengan cara: pada Insert tab, Illustration group, klik Insert Picture. Anda dapat memperoleh kumpulan gambar menawan di artikel ini: 20 Websites untuk Download Gambar Gratis. Klik tombol Save untuk menyimpan cover page. Jika ingin menyimpannya ke galeri agar dapat digunakan lagi, ikuti cara berikut: Pilih seluruh isi cover page. Pada Insert tab, klik Cover Page dan pilih Save Selection to Cover Page Gallery. Pada kotak dialog Create New Building Block, beri nama cover page di kotak Name dan klik OK. Untuk menghapus cover page: Pada Insert tab, klik Cover Page dan pilih Remove Current Cover Page.
