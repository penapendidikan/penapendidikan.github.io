6. Sebutkan fungsi tiga operator referensi dan berikan contohnya​


Jawaban:

Yang dimaksud operator pada microsoft excel adalah cara menuliskan elemen rumus atau formula.

Ada 4 tipe operator :

1) Operator aritmatika adalah cara menuliskan elemen rumus operasi matematika dasar.

a. + (tanda plus) untuk penambahan

b. +- (tanda minus) untuk pengurangan dan negasi

c. * (tanda bintang) untuk perkalian

d. / (garis miring) untuk pembagian

e. % (tanda persen) untuk persen

f. ^ (tanda sisipan) untuk perpangkatan

2) operator perbandingan adalah cara menuliskan elemen rumus atau formula untuk membandingkan dua nilai yang hasilnya adalah nilai logika TRUE atau FALSE.

a. = (tanda sama dengan) untuk sama dengan

b. > (tanda lebih besar dari) untuk lebih besar dari

c. < (tanda lebih kecil dari) untuk kurang dari

d. >= (tanda lebih besar dari atau sama dengan) untuk lebih besar dari atau sama dengan

e. <= (tanda lebih kecil dari atau sama dengan) untuk lebih kecil atau sama dengan

f. <> (tanda tidak sama dengan) untuk tidak sama dengan

3) operator penggabungan teks adalah cara menuliskan elemen rumus atau formula untuk menggabungkan satu atau beberapa string teks dengan tujuan menghasilkan sebuah satu nilai teks yang berkelanjutan.

operator penggabungan teks menggunakan & (dan).

4) operator referensi adalah cara menuliskan elemen rumus atau formula untuk menggabungkan rentang sel.

a. : (titik dua), disebut operator rentang, yang menghasilkan satu referensi di antara sel yang dipilih.

b. , (koma), disebut operator gabungan, yang menggabungkan beberapa referensi menjadi satu referensi.

c. (spasi), disebut operator irisan, yang menghasilkan referensi ke sel yang umumnya untuk dua referensi.

d. ; (titik koma), mempunyai fungsi yang sama dengan koma. untuk menggabungkan atau mengkombinasikan rentang referensi.

Berikut adalah penjelasan mengenai berbagai operator dalam bahasa pemrograman python

Operator matematis (arimatika)

Operator +

Operator + (plus) digunakan untuk menjumlahkan dua objek. Bila kedua objek bernilai numerik maka ia akan dijumlahkan secara numeris. Bila ia berupa teks (string), maka kedua objek tersebut akan digabung.

a = 5

b = 4

c = a + b # c akan bernilai bilangan bulat 9

a = 'Hello'

b = 'World'

c = a +' '+ b # c akan bernilai teks 'Hello World'

Operator –

Operator – (minus) digunakan untuk mengurangi objek dikiri dengan objek di kanan (seperti ekspresi matematis biasa). Jika objek dikiri tidak ada, ia dianggap 0. Operator – tidak dapat digunakan pada nilai teks (string)

a = 5

c = a - 2 # c akan terisi nilai 3

c = - a # c akan terisi nilai -5

Operator * dan **

Operator * (perkalian) digunakan untuk mengalikan dua buah objek, sedangkan operator ** (power) digunakan untuk memangkatkan objek.

a = 3

b = 2

c = a * b # c akan terisi nilai 6

d = a ** b # d akan terisi nilai 9

Operator /, //, dan %

Operator / (pembagi) digunakan untuk membagi objek. Jika yang digunakan adalah operator // maka pembagian akan dilakukan dengan pembulatan (floor). Operator % (modulus) akan menghasilkan sisa dari hasil pembagian.

Perhatikan contoh kode di bawah untuk lebih jelasnya.

a = 7

b = 3

c = a / b # c akan terisi nilai 2.3333

d = a // b # d akan terisi nilai 2

e = a % b # e akan terisi nilai 1

Operator relasional

Operator relasional adalah operator yang kita gunakan untuk menguji nilai kebenaran sebuah ekspresi, kebenaran berarti apakah sebuah ekspresi bernilai True atau False.

Berikut adalah daftar operator relasional dan maknanya:

Operator <, bermakna lebih kecil dari

Operator <=, bermakna lebih kecil atau sama dengan dari

Operator >, bermakna lebih besar dari

Operator >=, bermakna lebih besar atau sama dengan dari

Operator ==, bermakna sama dengan

Operator !=, bermakna tidak sama dengan

Dalam kode berikut disertakan contoh penggunaan operasional beserta hasil ekspresinya.

a = 7

b = 3

c = a < b # c akan terisi False

c = a <= b # c akan terisi False

adalah operator yang menggunakan tanda sama dengan (=) untuk mengisi sebuah nilai dalam suatu variabel.

int ezra = 11;

adalah operator yang digunakan untuk melakukan operasi dimana nilai yang dihasilkan dari operasi tersebut hanya berupa  nilai TRUE (benar) atau FALSE (salah). (&& - AND (dan), || - OR (atau), ! - NOT(negasi))
Contoh:
if (ezra == 10 && dove == 11) {
  print ("lovely");
} else if (ezra == 10 || dove == 11 {
  print ("ugly");
} else if (!ezra) {
  print ("haha")
}

ezra = 1 * 10 - 9;
dove = 10 / 2 + 5;

adalah operator yang digunakan untuk menentukan relasi atau hubungan dari dua buah operand. (< , > , >= , <= , == , !=)

if (ezra > 10) {
  print ("haha");
}
