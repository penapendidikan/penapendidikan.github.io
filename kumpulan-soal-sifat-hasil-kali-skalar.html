11. buatlah 5 contoh soal, beserta jawabannya tentang perkalian matriks dengan skalar?​


Jawaban:

Jika A dan B adalah matriks berordo m x n sedangkan k1 dan k2 adalah skalar, maka berlaku sifat perkalian skalar sebagai berikut:

k1 (A + B) = k1A + k1B

(k1 + k2) A = k1A + k2A

k1(k2A) = (k1k2) A

Jika A, B dan C adalah matriks berordo m x n dan k adalah skalar maka berlaku sifat-sifat perkalian matriks sebagai berikut:

A x B ≠ B x A

k x (A x B) = (k x A) x B

A x (B x C) = (A x B) x C

A x (B + C) = (A x B) + (A x C)

(A + B) x C = (A x C) + (B x C)

Jika perkalian hanya memuat matriks-matriks persegi, terdapat unsur identitas yaitu I sehingga AI = IA = A

Perkalian dengan matriks O, yaitu AO = OA = O

21. Untuk himpunan S = {(x,y,z,u) |2x - y + z - 2u = 2} dengan penjumlahan dan perkalian dengan skalar di R4. a. Periksa, apakah sifat ketertutupan operasi penjumlahan dan perkaliandengan skalar dipenuhi! b. Sebutkan definisi dan atau dalil mana saja yang digunakan dalam menyelesaikan soal pada a)​


Penjelasan dengan langkah-langkah:


a. Untuk memeriksa apakah sifat ketertutupan operasi penjumlahan dan perkalian dengan skalar dipenuhi, kita perlu memeriksa apakah hasil operasi tersebut masih tetap berada di dalam himpunan S.

Misalkan ada dua vektor (x1,y1,z1,u1) dan (x2,y2,z2,u2) dalam S, dan k adalah suatu skalar di R.

Penjumlahan:

(x1,y1,z1,u1) + (x2,y2,z2,u2) = (x1 + x2, y1 + y2, z1 + z2, u1 + u2)

Dari sifat-sifat operasi pada sistem persamaan linear, jika 2x - y + z - 2u = 2 dan 2x' - y' + z' - 2u' = 2, maka

2(x + x') - (y + y') + (z + z') - 2(u + u') = 2 + 2 = 4

Sehingga hasil penjumlahan tetap berada dalam himpunan S. Oleh karena itu, sifat ketertutupan operasi penjumlahan terpenuhi.

Perkalian dengan skalar:

k(x1,y1,z1,u1) = (kx1,ky1,kz1,ku1)

Dari sifat-sifat operasi pada sistem persamaan linear, jika 2x - y + z - 2u = 2, maka

2(kx) - (ky) + (kz) - 2(ku) = 2k(2x - y + z - 2u) = 2k(2) = 4k

Sehingga hasil perkalian dengan skalar tetap berada dalam himpunan S. Oleh karena itu, sifat ketertutupan operasi perkalian dengan skalar terpenuhi.

b. Definisi yang digunakan dalam menyelesaikan soal adalah sifat-sifat operasi pada sistem persamaan linear. Untuk penjumlahan, digunakan sifat bahwa hasil penjumlahan dari dua vektor yang memenuhi persamaan sistem linear juga memenuhi persamaan tersebut. Untuk perkalian dengan skalar, digunakan sifat bahwa hasil perkalian skalar dengan sebuah vektor yang memenuhi persamaan sistem linear juga memenuhi persamaan tersebut.
