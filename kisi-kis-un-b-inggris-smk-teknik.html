kaleng cat = x

kuas = y

• Eliminasi:

2x + 3y = 101.500 (×2)

x + 2y = 53.500 (×3)

_______________________

4x + 6y = 203.000

3x + 6y = 160.500

___________________(–)

x = 42.500

Diperoleh harga satu kaleng cat = Rp42.500

• Substisukan x:

x + 2y = 53.500

42.500 + 2y = 53.500

2y = 53.500 - 42.500

2y =11.000

y = 5.500

Diperoleh harga satu kuas = Rp5.500

•••••

Sehingga,

= 1 kaleng cat + 1 kuas

= 42.500 + 5.500

= Rp48.000,00

37. Kisi kisi un tolong bantu jawab , bukan pas lagi ujian tq


Tinggal menggunakan perbandingan saja.

%teknik mesin = 100%-30%-10%-45% = 15%

maka,

%teknik mesin = 15%

%TKJ = 30%

siswa TKJ = 150 orang

siswa teknik mesin?

[tex]\frac{siswa teknik mesin}{siswa TKJ}[/tex] = [tex]\frac{%teknik mesin}{%TKJ}[/tex]

[tex]\frac{siswa teknik mesin}{150}[/tex] = [tex]\frac{15%}{30%}[/tex]

siswa teknik mesin = [tex]\frac{150x15%}{30%}[/tex]

siswa teknik mesin = 75 orang
