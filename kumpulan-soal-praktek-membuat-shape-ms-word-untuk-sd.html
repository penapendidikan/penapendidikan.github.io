17. Apa saja icon yang dibutuhkan untuk membuat cover di ms word??​


beberapa icon yang mungkin dibutuhkan:

Icon Font: Untuk memilih jenis huruf dan ukuran huruf yang ingin digunakan pada cover.Icon Shape: Untuk menambahkan bentuk pada cover, seperti lingkaran atau kotak.Icon Picture: Untuk menambahkan gambar atau foto pada cover.Icon Text Box: Untuk menambahkan kotak teks pada cover.Icon Color: Untuk mengubah warna latar belakang atau warna pada teks atau bentuk.Icon Border: Untuk menambahkan garis tepi pada cover.Icon Page Layout: Untuk mengatur margin, ukuran kertas, dan orientasi halaman

18. Langkah langkah dalam menyisipkan gambar bangun (shape) pada ms word 2010


JAWABAN:Ikuti langkah - langkah dibawah ini!!

1.Buka aplikasi microsoft word 2010

2.Lalu klik insert yang ada diatas

3. Setelah klik insert,kemudian cari tulisan SHAPES,setelah itu di klik

4.Setelah kalian klik SHAPE akan banyak bentuk bentuk bangunan

5.Lanjut, kalian cari bentuk kubus / cube setelah itu di klik

6.Setelah kalian klik bentuk kubus itu akan muncul di kertas microsoft word itu

7.Lalu di bagian atas,bawah,kanan,dan kiri kubus masing masing memiliki 3 titik

8.Jika kalian ingin memindah mindahkan gambar klik bagian tengah kubus dan tahan

9.Lalu jika kalian ingin memanjangkan kubus klik saja titik yang ada diatas kubus tersebut,dan jika ingin melebarkan kubus klik saja titik yang ada dibawah kubus

10.Jika kalian ingin menyerongkan kubus,diantara 3 titik diatas balok itu akan ada titik berwarna hijau.Kalian klik saja tiitk hijau itu dan tahan,setelah itu kalian bebas membuat kubus itu menyerong ke kanan dan kiri.

19. sebuktankan.fungsi dari f1 sd f12. dalam ms. word buran ya fren soal nya lgi di butuhkan


1. F1Biasanya tombol ini digunakan untuk memanggil file help dari suatu software atau aplikasi.Tombol Win + F1 akan membuka Microsoft Windows help and support center.

2. F2Untuk di windows dapat digunakan sebagai jalan pintas mengubah nama suatu file maupun folder.Tombol kombinasi Alt + Ctrl + F2 digunakan untuk membuka dokumen baru di Microsoft Word.Ctrl + F2 menampilkan jendela print preview dalam Microsoft Word.

3. F3Sering kali diguanakan untuk membuka fitur pencarian untuk banyak program termasuk Microsoft Windows.Jika kalian sedang bekerja dalam Ms word atau yang lain dapat menggunakan CTRL+F atau mencari file di windows dapat menggunakan Win+FShift + F3 akan mengubah teks dalam Microsoft Word menjadi uppercase(huruf besar) atau   lowercase(huruf kecil) atau huruf kapital pada awal setiap kata.

4. F4Membuka jendela.Mengulangi aksi terakhir yang dilakukan di Word 2000 +Alt + F4 akan menutup program yang sedang aktif dalam Microsoft Windows.Ctrl + F4 akan menutup jendela yang terbuka dalam jendela aktif saat ini di Microsoft Windows.

5. F5Mungkin tombol ini yang paling populer untuk me reload halaman di browser maupun untuk refresh tampilan windows.Buka window pencarian, penggantian kata di Microsoft Word.Mulai tayangan slide show dalam PowerPoint.
 6. F6 Memindahkan kursor ke address bar di Internet Explorer dan Mozilla Firefox.

7. F7 Biasanya digunakan untuk memeriksa ejaan dan tata bahasa (spelling) dalam program-program Microsoft seperti Microsoft Word, Outlook, dllShift + F7 melakukan pemeriksaan Thesaurus pada kata yang disorot.Menyalakan tsobat sisipan browsing di Mozilla Firefox.

8. F8Fungsi tombol yang digunakan untuk masuk ke menu Windows startup, biasanya digunakan untuk masuk ke Windows Safe Mode.
 9. F9Membuka toolbar Pengukuran dalam Quark 5.0.
 10. F10Pada Microsoft Windows digunakan untuk mengaktifkan menu bar dari aplikasi yang terbuka.Shift + F10 adalah sama dengan klik kanan pada icon yang disorot, file, atau link internet.

11. F11Modus layar penuh di pada web browser.

12. F10 Membuka menu Save As pada Microsoft Word.Ctrl + Shift + F12 mencetak/print dokumen di Microsoft Word.F1:digunakan untuk membuka Microsoft windows
F12:untuk membuka menu save as pada microsoft word

23. Bagaimana cara Membuat tabel pada ms excel dan ms word !​


Jawaban:

Di Ms.Word :

1). Pastikan ribbon pada tab menu Insert.

2). Letakkan kursor di tempat dimana posisi tabel Excel akan dibuat.

3). Klik ikon menu Tabel lalu pilih Excel Spreadsheet. Insert, Tabel, Excel Spreadsheet.

Di Ms.Excel :

1). Pilih rentang data yang akan kamu jadikan sebagai tabel.

2). Selanjutnya, klik pada tab Insert lalu pilih Table. Atau tersedia shortcut keyboard dengan menekan tombol Ctrl + L atau Ctrl + T secara bersamaan.

3).Berikutnya akan muncul sebuah Option Box atau kotak pilihan "Create Table".
