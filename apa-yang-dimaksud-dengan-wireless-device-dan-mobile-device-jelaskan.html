4. Identifikasi dan jelaskan pengertian serta perbedaan antara input device,processing device dan storage device ​


Input Device, Processing Device, dan Storage Device adalah tiga komponen utama dalam sistem komputer. Berikut identifikasi dan penjelasan pengertian serta perbedaannya:

**1. Input Device (Perangkat Input):**

- **Pengertian:** Perangkat Input adalah perangkat keras yang digunakan untuk memasukkan data atau informasi ke dalam komputer. Ini berfungsi sebagai alat untuk berinteraksi dengan komputer, mengirimkan perintah, teks, gambar, atau suara ke sistem.

- **Contoh:** Keyboard, mouse, mikrofon, kamera web, pemindai (scanner), dan layar sentuh (touchscreen).

- **Fungsi:** Memasukkan data ke dalam komputer untuk diproses.

**2. Processing Device (Perangkat Pemrosesan):**

- **Pengertian:** Perangkat Pemrosesan adalah komponen utama yang bertanggung jawab untuk mengolah data dan menjalankan perintah. Ini merupakan otak komputer yang melakukan perhitungan dan pengolahan data.

- **Contoh:** CPU (Central Processing Unit) atau prosesor komputer.

- **Fungsi:** Melakukan pemrosesan data, perhitungan, dan menjalankan program.

**3. Storage Device (Perangkat Penyimpanan):**

- **Pengertian:** Perangkat Penyimpanan adalah perangkat keras yang digunakan untuk menyimpan data dan informasi dalam jangka waktu yang lebih lama. Ini digunakan untuk menyimpan program, file, dan data yang akan diakses nanti.

- **Contoh:** Hard disk drive (HDD), solid-state drive (SSD), USB flash drive, DVD, dan Blu-ray disc.

- **Fungsi:** Menyimpan data dan informasi agar dapat diakses kembali nanti.

**Perbedaan Utama:**

- **Input Device** digunakan untuk memasukkan data ke dalam komputer, sedangkan **Processing Device** melakukan pemrosesan data dan menjalankan program. **Storage Device** digunakan untuk menyimpan data.

- **Input Device** dan **Storage Device** adalah perangkat input dan penyimpanan, sementara **Processing Device** adalah perangkat yang memproses data.

- Contoh **Input Device** termasuk keyboard dan mouse, sedangkan **Processing Device** adalah CPU, dan **Storage Device** termasuk hard disk drive (HDD).

- **Input Device** mengirim data ke **Processing Device** untuk diproses, dan hasil pemrosesan tersebut kemudian dapat disimpan di **Storage Device**.

Ketiga komponen ini bekerja bersama-sama dalam sistem komputer untuk menerima, memproses, dan menyimpan informasi. Input Device memasukkan data, Processing Device mengolahnya, dan hasil pemrosesan dapat disimpan di Storage Device.
