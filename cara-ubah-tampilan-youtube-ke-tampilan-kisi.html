5. cara mengubah tampilan web menjadi tampilan html​


Jawaban:

Google Chrome ternyata bisa juga Anda manfaatkan untuk mengedit tampilan web Anda (mock up). Namun sebelum tutorial ini kami Lanjutkan, Anda sebaiknya harus sudah menguasai kedua hal berikut :

Dasar-dasar HTML

Dasar-dasar CSS Layout

Jika kedua hal tersebut belum Anda kuasai, sebaiknya anda membaca-baca dulu tentang css & html dari beberapa website berikut ini atau disini.

Perubahan Layout web yang bisa dilakukan dengan bantuan Google Chome sebenarnya menyangkut pengeditan CSS dan HTML secara live, sehingga langsung terlihat perubahannya di Layar. Anda bisa saja merubah warna dasar, menggeser atau merubah ukuran (panjang dan lebar) dari sebuah TAG.

Langkah untuk memulai perubahan Layout adalah sebagai berikut :

Buka website dengan Google Chrome

Tentukan Bagian yang akan anda Edit.

Klik kanan untuk memunculkan menu popup

inspect-element

Klik Inspect element

Muncul tampilan sebagai berikut:

chrome-web-developer-window

Tampilan di atas terdiri dari 3 bagian utama yaitu tampilan web, source code HTML dan CSS Style. Anda bisa mencoba melakukan editing di source HTML maupun di CSS Style. Untuk CSS style akan terlihat di baris mana editing dilakukan.

CSS-editing

Perubahan CSS pada Google Chrome sifatnya hanya untuk melihat perubahan secara live, tanpa merubah file aslinya. Jika Editing sudah OK, Anda bisa melakukan perubahan pada file asli.

Selamat mencoba

14. bagaimana cara mengubah tampilan pada diagram


Cara memformat diagram:

Klik sebuah diagram SmartArt pada dokumen untuk memilihnya.Pada tab kontekstual SmartArt Tools | Design, di grup SmartArt Styles, klik thumbnail pada galeri SmartArt Styles.Jika Anda ingin mengubah warna Theme yang digunakan oleh diagram SmartArt, pada tab kontekstual SmartArt Tools | Design, di grup SmartArt Styles, klik Change Colors. Klik thumbnail pada galeri yang menampilkan warna Theme.

￼


Cara mengubah tipe atau data diagram:

Klik sebuah diagram SmartArt pada dokumen untuk memilihnya.Jika Anda ingin mengubah layout dari diagram SmartArt, pada tab kontekstual SmartArt Tools | Design, di grup Layout, arahkan ke thumbnail yang ada pada galeri Layout dan lihat preview-nya. Klik pada thumbnail untuk mengaplikasikannya.Pada tab Design dibawah tab kontekstual SmartArt Tools, di grup Create Graphic, klik tombol Text Pane.

￼

Klik tiap placeholder dan masukkan teks yang ingin Anda tampilkan pada diagram SmartArt. Jika teks yang Anda masukkan terlalu panjang untuk ditampung dalam Shape, ukuran huruf teks akan otomatis dikurangi di semua Shape pada diagram SmartArt tersebut (tidak hanya di satu Shape).

21. Untuk mengubah tampilan slide caranya klik...


Cara yang dilakukan untuk mengubah tampilan slide pada microsoft powerpoint adalah dengan mengklik bagian menu Design. Pada menu Design terdapat 3 ribbons tabs yang dapat digunakan yaitu ribbons tabs design page setup, ribbons tabs design themes serta ribbons tabs design background.

Pembahasan

Slide yang ada pada microsoft powerpoint merupakan istilah untuk lembar kerja digunakan pada saat menggunakan aplikasi microsoft powerpoint. Slide yang ada pada microsoft powerpoint memiliki beberapa jenis yaitu:

27. Bagaimana cara mengubah tampilan kamera pada sketchup


Cara Merubah Background Dengan Warna

1.Pilih tab Window, klik pada style. Akan muncul kotak dialog style


2.Jika ingin merubah warna background pada area kerja, di tab select ganti ke In Model, selanjutnya pilih warna yang sesuai dengan keinginan

Cara Merubah Background Dengan Gambar

Pilih tab Window, klik pada style. Akan muncul kotak dialog stylePilih tab edit, Klik tanda plus (add watermark), kemudian cari gambar yang ingin, klik open

￼

Ganti pilihan background, klik next

￼

Atur blend-nya, yaitu tingkat transparansi pada area kerja, klik next

￼

Pada tahap ini anda dapat memilih salah satu settingan watermark yang diinginkan; Stretched to fit the screen (menempatkan satu gambar dalam layar kerja), Tiled across the screen (menampilkan gambar diseluruh bagian layar kerja dengan ukuran per piksel), dan positioned in the screen (menempatkan gambar pada area-area yang diinginkan seperti di pojok kanan.

Selanjutnya klik finish untuk mengakhiri
