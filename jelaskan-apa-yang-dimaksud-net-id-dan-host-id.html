3. perbedaan net id dan host id


Net ID berperan dalam identifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network. Jadi, seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama. Garis pemisah antara bagian network dan host tidak tetap, bergantung kepada kelas network. IP address dibagi ke dalam lima kelas, yaitu kelas A, kelas B, kelas C, kelas D dan kelas E.

9. Tentukan Host ID dan Net ID dari IP address 10.10.10.1​


Host ID dan Net ID adalah bagian-bagian dari alamat IP yang digunakan untuk membedakan antara perangkat atau host dalam jaringan dengan jaringan itu sendiri.

Untuk IP address 10.10.10.1, dapat ditentukan sebagai berikut:

Net ID: 10.10.10Host ID: .1

Net ID merupakan bagian dari alamat IP yang digunakan untuk menentukan jaringan yang digunakan, dan Host ID merupakan bagian dari alamat IP yang digunakan untuk menentukan perangkat yang terhubung ke jaringan tersebut.

Net ID dapat digunakan untuk menentukan subnet dari jaringan, sementara Host ID digunakan untuk menentukan alamat unik dari perangkat yang terhubung ke jaringan tersebut.


10. Berapa net id dan host/subnetnya dari IP 192.168.200.200/24


Berikut adalah langkah-langkah untuk menghitung net id dan host/subnet:

- Ubah IP dan subnet mask ke bentuk biner. Anda dapat menggunakan kalkulator biner¹ untuk membantu Anda.

IP: 192.168.200.200 = 11000000.10101000.11001000.11001000

Subnet mask: /24 = 255.255.255.0 = 11111111.11111111.11111111.00000000

- Lakukan operasi logika AND antara IP dan subnet mask untuk mendapatkan net id.

Net id: 11000000.10101000.11001000.11001000 AND 11111111.11111111.11111111.00000000

= 11000000.10101000.11001000.00000000

= 192.168.200.0

- Balikkan bit dari subnet mask dan lakukan operasi logika AND lagi antara IP dan subnet mask yang dibalik untuk mendapatkan host/subnet.

Subnet mask yang dibalik: /24 = 255.255.255.0 = 11111111.11111111.11111111.00000000

= 00000000.00000000.00000000.11111111

Host/subnet: 11000000.10101000.11001000.11001000 AND 00000000.00000000.00000000.11111111

= 00000000.00000000.00000000.11001000

= 0.0.0.200

Dengan demikian, net id dari IP tersebut adalah 192.168.200 dan host/subnetnya adalah 0.

32. Jelaskan apa yang dimaksud dengan Network id dan host id dan berikan contohnya


Pada dasarnya Network ID sendiri dapat kita artikan sebagai suatu Identitas dari suatu kelompok IP Address atau subnet, yang mana Network ID ialah suatu bagian dari IP address yang menunjukan letak network atau jaringan dari suatu IP Address. Sedangkan untuk Host ID sendiri ialah suatu nomor perangkat keras fisik Ethernet Network Interface Card atau yang sering disingkat dengan sebutan NIC.

Sebagai salah satu contohnya ialah:

Kelas A yaitu 10.11.12.1 maka untuk contoh ini network ID nya ialah 10. sedangkan untuk Host ID ialah 11.12.1.Kelas B yaitu 188.10.11.1 maka untuk contoh ini network ID nya ialah 188.10. sedangkan untuk Host ID ialah 11.1.Kelas C yaitu 192.168.1.1 maka untuk contoh ini network ID nya ialah 192.168.1 sedangkan untuk Host ID ialah 1.

Pembahasan

Pada umumnya Net ID sendiri dapat berperan penting dalam mengidentifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network, yang mana pada konteks ini dapat kita simpul kan bahwa seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama.
